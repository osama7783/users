# 🔐 Users – Full Stack Authentication & User Management System

## 📌 Project Overview

**Users** is a full-stack authentication and user management system built using **Java Spring Boot**, **ReactJS**, and **PostgreSQL**.

The system implements a secure authentication architecture using **JWT (Access & Refresh Tokens)**, along with **Role-Based Access Control (RBAC)** and full **user profile management**.

This project demonstrates real-world backend security design and full-stack integration between frontend and backend systems.

---

## 🚀 Features

- 🔐 JWT Authentication (Access & Refresh Tokens)
- 📝 User Registration & Login system
- ♻️ Secure Token Refresh mechanism
- 👤 User Profile Management (view & update)
- 🧑‍💼 Admin User Management (CRUD operations)
- 🌐 Role-Based Access Control (USER / ADMIN)
- ⛔ Custom 403 Forbidden handling
- ❌ Custom 404 Not Found page
- 🔄 Protected routes for authenticated users
